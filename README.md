# 配色方案 ColorScheme

## 功能

支持切换和导入文字配色文件，独立于主题。安装完成后，在插件系统设置左侧找到配色方案，进行使用即可。配色文件格式为json
```json
{
    "name": "avatar",
    "schemes": {
        "color1": "#0088cc",
        "color2": "#0088cc",
        "color3": "#0088cc",
        "color4": "#0088cc",
        "color5": "#0088cc",
        "color6": "#0088cc",
        "color7": "#0088cc",
        "color8": "#0088cc",
        "color9": "#0088cc",
        "color10": "#0088cc",
        "color11": "#0088cc",
        "color12": "#0088cc",
        "color13": "#0088cc",
        "background1": "#0088cc",
        "background2": "#0088cc",
        "background3": "#0088cc",
        "background4": "#0088cc",
        "background5": "#0088cc",
        "background6": "#0088cc",
        "background7": "#0088cc",
        "background8": "#0088cc",
        "background9": "#0088cc",
        "background10": "#0088cc",
        "background11": "#0088cc",
        "background12": "#0088cc",
        "background13": "#0088cc"
    }
}
```
